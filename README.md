Internet Programming course project

Goal of this project was to make website of faculty department with following use cases:
  1. Administrator:
      -  Add/modify/delete user
      - Add general notification
      - Add connection between courses and lecturers
  2. Student:
      - Choose courses to follow
  3. Lecturer:
      - Update personal informations
      - Update course page content and notifications
  4. General:
      - View list of courses
      - View list of lecturers and their personal pages
      - View general informations about deparment
